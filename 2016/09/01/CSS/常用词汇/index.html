  1. A
    1. absolute [‘æbsəlu:t] 绝对的，完全的
    2. active [‘æktiv] 活动的，激活的，标记的一个伪类
    3. align [ə’lain] 对齐

    1. background [‘bækgraund] 背景

    3. black [blæk] 黑色
    4. blink： [bliŋk] 闪亮,闪烁
    5. block [blɔk] 块
    6. blue [blu:] 蓝色
    7. body [‘bɔdi] 主体，一个HTML标记
    8. bold [bəuld] 加粗
    9. border [‘bɔ:də] 边框、框线 边框
    10. both [bəuθ] 两者(都);两个(都);是clear属性的一个属性值
    11. bottom [‘bɔtəm] 底;底部，是一个CSS属性
    12. box [bɔks] 箱;盒;匣
    13. br 换行标记
    14. bug [bʌg] 程序设计上的错误，漏洞等
    15. building [‘bildiŋ] 建筑
    16. button [‘bʌtn] 按钮

    1. cell [sel] 表格中的单元格
    2. center [‘sentə] 居中
    3. centimeter [‘senti,mi:tə] 厘米

    7. clear [kliə] 清除

    13. crosshair 十字准线
    14. css 层叠样式表
    15. cursor [‘kə:sə] 鼠标指针

    1. dashed [dæʃt] 虚线CSS边框常用的一种
    2. decimal [‘desiməl] 十进制
    3. decoration [,dekə’reiʃən] 装饰
    4. default [di’fɔ:lt] 默认的
    5. definition [,defi’niʃən] 定义
    6. design [di’zain] 设计
    7. display [di’splei ] 显示，CSS的一个属性
    8. division [di’viʒən] 分区，div就是这个单词的缩写
    9. document [‘dɔkjumənt] 文件文档
    10. dotted [‘dɔtid] 点线
    11. double [‘dʌbl] 双线
  5. E
    1. father [‘fɑ:ðə] 父亲,父
    2. filter [‘filtə] 过滤层；滤器
    3. first [fə:st] 第一个
